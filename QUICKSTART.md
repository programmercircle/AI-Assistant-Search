# Quick Start Guide

## Installation (3 steps)

### Step 1: Prepare the Extension
Download all the files to a folder named `ai-search-extension`

### Step 2: Load into Chrome
1. Open Chrome
2. Go to: `chrome://extensions/`
3. Turn ON "Developer mode" (top-right toggle)
4. Click "Load unpacked"
5. Select the `ai-search-extension` folder

### Step 3: You're Done!
The extension is now installed and ready to use.

## How to Use

### Simple 4-Step Process:

1. **Select** any text on a webpage
2. **Right-click** the selected text
3. Click **"Search with AI"**
4. Choose your AI assistant (Claude, ChatGPT, Grok, or Perplexity)

The AI service will open in a new tab with your text already filled in!
